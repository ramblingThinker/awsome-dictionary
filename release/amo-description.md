# AMO Store Description

## Summary (short)
Highlight a word on any webpage to get a clean, fast dictionary definition popup.

## Description (long)
Awesome Dictionary helps you understand words instantly while browsing.

- Double-click any word to open an in-page definition bubble.
- Right-click selected text and choose `Define '%s'`.
- Search words directly from the toolbar popup.
- View recent lookup history in the popup.
- Works with offline resilience: if the API is temporarily unavailable, previously saved definitions can be shown from local cache.
- Includes configurable settings for bubble timeout, auto-close behavior, history size, and context menu toggle.

No account required.
